# Movie Recommendation System

A Java-based movie recommendation system with:
- Object-oriented design (inheritance, polymorphism)
- Strategy pattern for recommendation algorithms
- JavaFX GUI
- CSV file persistence

## Features
- User registration and login
- Browse movies by genre/year/rating
- Personalized recommendations (Genre/Rating/Year based)
- Premium user support

## Technologies
- Java
- JavaFX
- Strategy Pattern
- File I/O
